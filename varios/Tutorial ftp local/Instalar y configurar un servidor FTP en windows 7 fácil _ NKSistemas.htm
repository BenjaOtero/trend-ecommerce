INICIO Infraestructura Instalar y configurar un servidor FTP en windows 7 fácil

Instalar y configurar un servidor FTP en windows 7 fácil

  • Infraestructura
  • Windows 7
may 14, 2012
13 17877

Vamos a ver como se instala y configura un servidor FTP en Windows 7 de forma simple, también nos va a servir para colocar una web, la idea es probar y conocer un poco más las herramientas de windows.

No hace falta instalar nada, aunque no lo creas, windows trae las herramientas necesarias solo que hay que activarlas.

1- Para ello vamos al panel de control, programas, activar o desactivar las características de windows. O simplemente hacemos inicio, en buscar programas o archivos colocamos “Activar o desactivar las características de windows”

Ahora hay que comprobar que el servicio este corriendo, por las dudas, vamos a inicio ejecutar services.msc y buscamos el servicio que dice, “Servicio FTP de Microsoft” boton derecho y lo iniciamos, si esta activo continuamos con la guía.

–

2- Una vez que se cargue los programas, vamos al panel de control de nuevo y en herramientas administrativas, seleccionamos “Administrador de internet information Services (IIS)” o lo buscamos más rápido desde inicio, buscar programas y archivos.

Y le damos botón derecho al nombre del sitio y agregamos un sitio FTP, como vemos también se puede instalar un sitio web, es muy similar.

–

3- Le vamos a asignar en este caso una IP al nombre y vamos a crear una carpeta que este en el disco C: y se llame FTP.

–

4- Seleccionamos puerto y si necesitamos SSL.

–

5- En esta sección elegimos la forma de autenticación, y podemos seleccionar usuarios específicos.

–

6- Vamos a comprobar que este bien configurado, así que abrimos un navegador y colocamos el nombre del servidor, en este caso la dirección IP. Previamente colocamos un archivo de texto en la carpeta principal en nuestro ejemplo C:FTP. También se puede configurar algún cliente FTP como filezilla o FireFTP que es el complemento de firefox.

–

Listo, así de simple. Saludos
